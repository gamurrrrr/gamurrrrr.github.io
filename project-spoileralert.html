The Challenge

Building a review platform requires more than just displaying text. It needs secure user authentication, protection against bad actors (spam/harassment), and an efficient way to search through thousands of movie records.

The Solution

We built a Python-based application using a JSON database for flexibility. The system features Role-Based Access Control (Admin vs. User), Regex password security, and a robust search engine capable of filtering by genre, director, and actor.

System Architecture & User Flow

1. Secure Authentication

Security was a "High Priority" requirement. We implemented Regex Validation to enforce strong passwords (8+ chars, numbers, symbols) and encrypted storage to prevent data leaks.

3. Admin Moderation

To maintain community safety, we built a dedicated Admin Dashboard separate from the standard user view.

Admin Capabilities:

  • View list of reported/flagged reviews.
  • Delete inappropriate content permanently.
  • Grant or revoke privileges for other users.

Agile Process & Traceability

The Traceability Matrix

We used a Requirements Traceability Matrix (RTM) to ensure every User Story was directly linked to a specific Test Case and a Pull Request (PR). This eliminated "scope creep" and ensured 100% test coverage for critical features like Account Creation and Review Deletion.
